Piano Tile Bot

Sept, 30, 2019

Python

Goal: Create a program that automatically sees and clicks piano tiles in the piano tile game to get the highest score possible.

Main Challange: Making PyAutoGUI find and click the piano tiles fast enough and speed up as the tiles speed up. Also a weird 32bit-64bit error :(

Main Learnings: A lot about PyAutoGUI.

Modules Used: PyAutoGUI

I have always found breaking games fun. Because it is, it really and truly is. That is why this project was so rewarding because I broke piano tiles. If you don’t know what the game is, piano tiles let me explain. It is basically a game where black squares move down on a screen and you have to click them. When you click them, they make a sound that goes with a song.
To break this particular game I used the super-powerful python module PyAutoGUI. There are many, many things you can do with PyAutoGUI, I used the function that is able to ‘see’ what color a specific pixel on the screen is. Using this functionality, I constantly looped, checking the four spots where black tiles would show up. If one of the if statements evaluated to true that specific spot where PyAutoGUI saw the black pixel of the tiles would be clicked. The same goes for the other three spots a tile could show up. To account for the game speeding up I had an up factor variable that makes the mouse click just a bit higher every time.
This was a blast to create and I am pumped to find another game to make a bot for.
