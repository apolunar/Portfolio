Check Grades

Nov, 9, 2019

Python

Goal: Create a program that scrapes the website my school uses to show students their grades and sends my grades and missing assignments to me in an email.

Main Challange: Getting the validation for the website my school uses to show students their grades using the module requests.

Main Learnings: I learned a lot about scraping websites and how to send an email with Python.

Modules Used: requests, lxml, smtplib

As the header of my portfolio says, I am still in school and still learning so I want to stay on top of my grades. I always find it annoying signing into my school’s web app that allows students to see their grades. I know, first world problem. Anyhow, I also find it annoying after logging in finding out that I have missing assignments that are sitting right in my backpack.
To fix this problem I enlisted the help of the web for some answers. There were a few things that needed to happen for everything to come together. The first being the login, I can’t get my grades if I don’t have access to the data. The second piece of the puzzle is reading the data, because after having access to the data you need to read it. Third, send an email from me to myself containing all of the required info (class, grade, missing assignments).
To figure out how to do this I started to research and I found a python module that satisfies the first requirement; login. This was done with the modules requests and lxml by finding the event token, view token, and viewstate token using xpath; which are required to login and plugging them into a payload dictionary. Using my provided username, password, and the event token, view token, and viewstate token I was able to successfully login and navigate to the grade book URL. From there I used lxml to find all of the course titles, grades, and missing assignments and passed the info to a sendemail function. The sendemail function uses smtplib to send the email which worked great!
But I wasn’t satisfied because the information passed to the sendemail function was sloppy and hard to read. So I took one step further recompiling the information using a for loop and HTML to make different pieces of info different colors and on separate lines. I also made an if statement that checked if there was more than one missing assignment and if so changed the missing assignment color to red. After that, the email looked beautiful and sent me valid information.
The one thing missing that I have to re-setup is a server to automatically send the email to me before I go to school and I plan on doing that later. This project was quite educational and extremely difficult. I truly am glad I put the time in because it did help me stay on top of my grades.
